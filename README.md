# OAT — Observatorio Ambiental Territorial

Plantilla base: Arctic Minimal.

Esta versión incorpora:
- Identidad OAT.
- Logo horizontal único en la barra superior.
- Eliminación de los dos espacios de logos anteriores.
- Paleta coordinada con el logotipo: azul marino, verde ambiental y azul territorial.
- Navegación lateral persistente.
- Páginas de difusión, plataforma, aprendizaje QGIS/ArcGIS Pro, capas y recursos, equipo y contacto.
